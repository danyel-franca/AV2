# Instruções 
1. pip install django -> Comando para instalar o django.
2. django-admin startproject nome_do_projeto . -> Cria um novo projeto django.
3. python manage.py runserver -> Roda o servidor.
4. python manage.py migrate -> Aplica as migrações pendentes.
5. python manage.py createsuperuser -> Criação do super usuário.
6. python manage.py startapp nome_do_app-> Criação de um app
